# Crayon
Colors things in minecraft 1.19.2 :D

```>``` is for green

```<``` is for red

```"`"``` is for blue

```:``` is for Light blue

```!``` is for yellow

```@``` is for pink

USAGE

add any of those symbols before your msg to color it
